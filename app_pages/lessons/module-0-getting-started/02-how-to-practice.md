# How to Practice SQL

The fastest way to learn SQL is to run queries yourself. Reading about a keyword is one thing; watching it return rows is another.

## The sample database

This course uses a small sample database with fake data about **users** and **payments**. Every example you'll see has been tested against it, so you can follow along and get the same results.

## Running queries without installing anything

You don't need to set up a database engine to start. A simple option is the free online tool [sqlime.org](https://sqlime.org/):

1. Download the sample database file (`db.sqlite`).
2. Open sqlime.org and use the **"open file"** button to upload it.
3. Start typing queries.

That's enough to work through every lesson in this course.

> If you prefer a local setup, any SQLite client works too. The exact tool matters less than getting into the habit of running each example and tweaking it.

## A note on databases

SQL is a standard, but every database (SQLite, MySQL, PostgreSQL, Snowflake, BigQuery…) has its own dialect. Most of what you learn here is universal. Where a function differs between databases, the lessons will call it out — and the official documentation for your database is always the final word.
